Debug:
XDU_DBG - set to 1 for debug support

Logging:
XDU_LOG - set to 1 for logging support
XDU_LOG_TTY - set to 1 to log to STDERR
XDU_LOG_FILE - set to 1 to log to ${PWD}/xdu.log
XDU_LOG_ERR_LVL - set to 1 to choose highest logging level of log records to log (less log records)
XDU_LOG_DBG_LVL - set to 1 to choose lowest logging level of log records to log (more log records)
